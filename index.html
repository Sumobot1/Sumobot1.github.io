Hi, I'm Michael!

Welcome to my website! I am a University of Waterloo student in term 1A of Mechatronics Engineering. I consider this website to be a side project of sorts, and a way to showcase my relevant experience, abilities, and interests. This site is very much a work in progress!

My projects include a universal television remote, a light board that flashes to music, a desktop game written in Java, and the computer science contest questions I have completed. I have a link to my Github, where I have the code for the projects that I have worked on. My website is written in HTML, CSS, and Javascript using a framework called Bootstrap.
